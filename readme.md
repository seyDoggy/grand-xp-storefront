# Grand-XP Storefront

Grand-XP Storefront is a custom child theme of the [Storefront](https://woocommerce.com/storefront/) theme, tailored specifically for Grand Experiences. It is designed to extend the functionality and styling of the parent theme. 

**Note:** This theme operates entirely independent of the WooCommerce plugin and instead utilizes FareHarbor for booking integrations and custom functionality.

## Installation

1. Ensure you have the parent [Storefront](https://woocommerce.com/storefront/) theme installed. (The WooCommerce plugin is not required).
2. Download or clone this child theme into your WordPress `wp-content/themes` directory.
3. Activate the Grand-XP Storefront theme from the WordPress Appearance dashboard.

## Customization

To customize the theme, you can:

- Edit the `style.css` file for custom global and block-specific styles.
- Modify the `functions.php` file to add or override functionality, including FareHarbor CTA integrations and custom block editor styles.
- Use the WordPress Customizer for additional structural settings.
